Functions

Reusable procedures

  • Functions allow us to write reusable, modular code.
  • We define a "chunk" of code that we can then execute at a later point.
  • We use them ALL THE TIME

Function Expressions

  • function add(x, y) {
    return x + y;
    }
  • const add = function (x, y){
    return x + y;
    }

Try/Catch

function yell(msg) {
try {
console.log(msg.toUpperCase().repeat(3))
} catch (e) {
console.log(e);
console.log("Please pass a string next time!")
}
}

How a map and forEach functions would look like:

(Provided this just to understand how foreach and map function definitions would look like)

ForEach:

  1. Array.prototype.forEach1= function(callback){
  2. for (let i = 0; i < this.length; i++) {
  3.     callback(this[i], i, this);
  4.   }
  5. }

Map:

  1. Array.prototype.map1 = function(callback){
  2. const newarray = [];
  3. for (let i = 0; i < this.length; i++) {
  4.     newarray.push(callback(this[i], i, this));
  5.   }
  6.   return newarray;
  7. }
